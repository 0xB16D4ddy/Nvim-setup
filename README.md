# Project Configuration

## Prerequisites

- **Neovim** >= 0.11
- **Make**
- **Delta**
- **ripgrep** (telescope, good for vimgrep replacement)
- **fd** (telescope)

## Utilities

- **Wakatime**
- **Chezmoi**

## Note

If NvChad UI's color is broken, use the command: `:NvChadLoadAllHighlights`
